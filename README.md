# Student Search with Array.find()

A simple JavaScript exercise demonstrating the usage of `Array.prototype.find()` to search for a student by name.

## Features
- Real-time search (on button click)
- Case-insensitive name search
- Clean & minimal interface

## Technologies Used
- HTML
- CSS
- Vanilla JavaScript (Array.find method)

## How to Use
1. Clone or download the project
2. Open `index.html` in your browser
3. Type a student name in the input field
4. Click **SEARCH** button
